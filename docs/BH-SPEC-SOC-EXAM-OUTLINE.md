# BH-SPEC-SOC: Specialisation — SOC & Incident Response — Exam Outline

**Course:** BH-SPEC-SOC

## Alignment

This course deepens **CySA+** Security Operations and Incident Response in a SOC context. Content is aligned to CySA+ domains 1–4 at an advanced level (SIEM, threat hunting, incident handling). No CASP+/SC-200/AWS in alignment per product decision; CySA+ only.
