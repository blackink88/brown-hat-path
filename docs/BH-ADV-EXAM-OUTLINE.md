# BH-ADV: Advanced & Leadership — Exam Outline

**Course:** BH-ADV  
**Primary certs:** CISSP, CISM, TOGAF (security/EA)

## CISSP — 8 Domains (overview)

1. Security and Risk Management  
2. Asset Security  
3. Security Architecture and Engineering  
4. Communication and Network Security  
5. Identity and Access Management  
6. Security Assessment and Testing  
7. Security Operations  
8. Software Development Security  

## CISM — 4 Domains

1. Information Security Governance  
2. Information Risk Management  
3. Information Security Program  
4. Incident Management  

## Module → Exam Mapping (BH-ADV)

| Module | Lessons | CISSP | CISM |
|--------|---------|-------|------|
| Security and Risk Management | Governance and Risk, Security Program Development | Domain 1, 5 | Gov, Risk, Program |
| Security Architecture and Leadership | Architecture and Engineering, Leadership and Communication | Domain 3, leadership | Program, Incident Mgmt |

## Coverage note

BH-ADV is strategic and leadership-focused. Use this outline when adding or editing lessons. Full CISSP/CISM prep typically requires additional study; this course provides the high-level alignment and leadership communication skills.
